# Local Image Search Engine

完全オフラインで動作する、ローカル環境向けのAI画像自動解析・高度検索アプリケーションです。
指定したフォルダ内の画像をAI（ConvNeXt / MOAT / VGG16）が自動解析してタグ付けや画風抽出を行い、CLIPモデルやFAISSを用いたベクトル検索と独自のスコアリングにより、日本語や直感的なワード、あるいは「好みの絵柄」で目的の画像を探し出すことができます。

## ✨ 主な特徴

* **完全オフライン動作**: Hugging FaceやTransformersの通信を遮断し、プライバシーを完全に保護します。
* **高度なAIタグ付け**: 画像処理に特化した2つのモデル（ConvNeXt, MOAT）によるアンサンブル推論で、高精度なタグ付けを実行します。
* **セマンティック検索**: 「10万語規模の翻訳辞書」と「CLIPモデルによるベクトル検索」を組み合わせることで、あいまいな表現でも目的の画像をヒットさせます。
* **独自スコアリング**: 検索ワードとの「類似度」× AIの「確信度」× タグの「希少度(IDF)」を掛け合わせ、最適な順序で結果を表示します。
* **リッチなUI**: Flet（Flutterベース）を採用。ピンチ操作によるグリッド変更、スワイプ操作対応の画像ビューアなど、スマホを通して快適に操作できます。
* **幅広いハードウェア対応**: DirectMLに対応しており、最新のGPU（RTX 50シリーズなど）でも煩雑なCUDA環境構築を回避しつつ、安定してGPU推論を動作させることが可能です。

## 🛠️ 環境構築

### 1. 必要なライブラリのインストール
Python環境（3.10以上推奨）で、以下のパッケージをインストールしてください。

```bash
pip install flet opencv-python pandas numpy faiss-cpu
pip install torch torch-directml torchvision
pip install onnxruntime-directml huggingface-hub transformers
pip install translators pillow

```

*(※ CUDA環境を使用する場合は、`torch-directml`や`onnxruntime-directml`を対応するCUDA版に置き換えてください)*

### 2. ディレクトリ構成

実行前に、画像を入れるためのフォルダを準備します。デフォルトでは `data/images` フォルダ直下、およびサブフォルダをスキャン対象とします。事前にフォルダを作成し、検索したい画像を配置してください。

```text
project_root/
├── app.py                 # UI・メイン実行ファイル
├── scripts/               # 初回セットアップ
│   ├── download_assets.py # 翻訳辞書ダウンローダー
│   ├── vectorize_tags.py  # タグのベクトルインデックス作成
│   └── sort.py            # 画像の重複排除と年月フォルダへの自動整理 (任意)
├── tasks/                 # アプリの裏で実行されるバックグラウンド処理群
│   ├── index.py           # 初期スキャン・サムネイル生成
│   ├── tagger.py          # 画像解析・タグ付け
│   └── vectorize_images.py# 絵柄(画風)ベクトルの生成
├── core/                  # バックエンドロジック
│   ├── database.py        # SQLite (FTS5) 制御
│   ├── search.py          # 検索全体のオーケストレーター
│   ├── tag_search.py      # タグのテキスト・ベクトル検索エンジン
│   └── style_search.py    # 絵柄(画風)のベクトル検索エンジン
├── ui/                    # UIコンポーネント
│   ├── search_bar.py      # 検索窓
│   ├── gallery.py         # グリッド表示
│   ├── viewer.py          # 画像拡大・詳細表示
│   └── drawer.py          # ブックマーク一覧

    ├── db/                # index.dbが生成されます
    ├── faiss/             # 絵柄検索用のインデックスが生成されます
    └── thumbnails/        # 縮小画像が生成されます

```

## 🚀 使い方 (初期セットアップ)

初回のインデックス構築手順です。順番にスクリプトを実行してください。

### Step 0: 画像の整理と重複排除 (任意)

iPhoneなどから取り込んだ雑多な画像群を、撮影年月（YYYYMM）ごとのフォルダに自動で振り分け、ファイル名を「撮影日時_ハッシュ値」にリネームして整理します。これにより、物理的な画像の重複を完全に排除し、将来的なマージも容易になります。

**💡 安全なコピー処理**
このスクリプトは元の画像（`data/images/` など）を移動するのではなく、新しいフォルダ（`data/sorted/` など）へ「コピー」する仕様になっています。そのため、数万枚の処理中に万が一エラーで中断されたりPCの電源が落ちたりしても、**元のデータが消えたり破損したりする心配はありません。**
また、再実行した際もハッシュ値によって既にコピー済みの画像は自動でスキップされるため、安心して続きから再開できます。

※実行前に scripts/sort.py 内の SOURCE_DIR（元の画像場所）と TARGET_DIR（整理後の保存先）が、自身の環境（data/images など）を指しているか確認してください。

```bash
python -m scripts.sort

```

### Step 1: 辞書のダウンロード

10万語規模の英日翻訳辞書を取得します。（初回のみネット接続が必要です）

```bash
python -m scripts.download_assets

```

### Step 2: タグリストのベクトル化

AIの出力タグを意味のベクトルに変換し、検索用のインデックスを作成します。（初回のみネット接続が必要です）

```bash
python -m scripts.vectorize_tags

```

### Step 3: 画像のスキャンとサムネイル生成

`data/images` フォルダ内の画像をデータベースに登録し、表示用の軽いサムネイルを作成します。

```bash
python -m tasks.index

```

### Step 4: AIによる画像解析（タグ付け）

データベースに登録された画像に対して、AIが自動で内容を判別しタグを付与します。
*(※ 枚数が多い場合は時間がかかります。DirectMLによるGPU支援が効いているかターミナルで確認してください)*

```bash
python -m tasks.tagger

```

### Step 5: 絵柄ベクトルの生成

VGG16モデルを使用して、全画像から画風（平均・標準偏差）を抽出し、検索用のFAISSインデックスを作成します。

```bash
python -m tasks.vectorize_images

```

## 🎨 アプリの起動と操作

すべての準備が整ったら、以下のコマンドでアプリを起動します。（完全オフラインで起動します）

```bash
python app.py

```

ブラウザが起動し、ローカルサーバー（デフォルト: `http://localhost:8000`）でUIが表示されます。

### 基本操作

* **検索窓**: 日本語、英語、俗語など好きな言葉を入力してEnterを押します。`-`（ハイフン）を先頭に付けることで**除外検索（マイナス検索）**も可能です。
* **絵柄検索**: style:作成した絵柄名 を入力すると、その画風に近い画像で絞り込めます。タグ検索と併用可能です（例: style:my_favorite 黒髪）。
* **サジェスト**: 文字を入力すると、DB内に存在するタグや辞書のヒット件数がサジェスト表示されます。絵柄名(style:)のサジェストにも対応しています。
* **ギャラリー**: 画面上の**ピンチイン・ピンチアウト**で、直感的に画像の列数（3〜6列）を変更できます。
* **絵柄タグの作成**: ギャラリー右上のボタンで「選択モード」に切り替え、複数枚の画像を選択して好きな名前を付けることで、オリジナルの絵柄タグを生成できます。

* **お気に入り**: サムネイル右下、またはビューア内のハートマークで画像を「お気に入り」に登録できます。検索窓が空の状態で検索を実行すると、お気に入り一覧が表示されます。
* **ブックマーク**: 検索窓のブックマークアイコンから、現在の検索条件を保存できます。画面右スワイプでドロワーを開き、保存した条件をすぐに呼び出せます。

### ビューア操作 (画像クリック時)

* **ダブルタップ**: 等倍 ⇔ 2倍ズームの切り替え。
* **ピンチ操作**: 自由な倍率での拡大・縮小とドラッグ移動。
* **スワイプ (左右)**: 前後の画像へ移動します。
* **スワイプ (上)**: 画面下部から「ファイル名」「ファイルパス」「付与されたタグ一覧」の詳細パネルを引き出します。
* **スワイプ (下)**: 詳細パネルを閉じる、またはビューア自体を閉じます。
