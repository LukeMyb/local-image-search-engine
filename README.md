# Local Image Searcher

完全オフラインで動作する、ローカル環境向けのAI画像自動解析・高度検索アプリケーションです。
指定したフォルダ内の画像をAI（ConvNeXt / MOAT）が自動解析してタグ付けを行い、CLIPモデルを用いたベクトル検索と独自のスコアリングにより、日本語や直感的なワードで目的の画像を探し出すことができます。

## ✨ 主な特徴

* **完全オフライン動作**: Hugging FaceやTransformersの通信を遮断し、プライバシーを完全に保護します。
* **高度なAIタグ付け**: 画像処理に特化した2つのモデル（ConvNeXt, MOAT）によるアンサンブル推論で、高精度なタグ付けを実行します。
* **セマンティック検索**: 「10万語規模の翻訳辞書」と「CLIPモデルによるベクトル検索」を組み合わせることで、俗語やあいまいな表現でも目的の画像をヒットさせます。
* **独自スコアリング**: 検索ワードとの「類似度」× AIの「確信度」× タグの「希少度(IDF)」を掛け合わせ、最適な順序で結果を表示します。
* **リッチなUI**: Flet（Flutterベース）を採用。ピンチ操作によるグリッド変更、スワイプ操作対応の画像ビューアなど、スマホを通して快適に操作できます。
* **ハードウェアアクセラレーション**: DirectMLに対応しており、幅広いGPUで高速な推論が可能です。

## 🛠️ 環境構築

### 1. 必要なライブラリのインストール
Python環境（3.10以上推奨）で、以下のパッケージをインストールしてください。

```bash
pip install flet opencv-python pandas numpy faiss-cpu
pip install torch torch-directml torchvision
pip install onnxruntime-directml huggingface-hub transformers
pip install translators pillow

```

*(※ CUDA環境を使用する場合は、`torch-directml`や`onnxruntime-directml`を対応するCUDA版に置き換えてください)*

### 2. ディレクトリ構成

実行前に、画像を入れるためのフォルダを準備します。デフォルトでは `data/images` フォルダ直下、およびサブフォルダをスキャン対象とします。事前にフォルダを作成し、検索したい画像を配置してください。

```text
project_root/
├── app.py                 # UI・メイン実行ファイル
├── core/                  # バックエンドロジック
│   ├── database.py        # SQLite (FTS5) 制御
│   ├── tagger.py          # 画像解析・タグ付け
│   ├── vectorize_tags.py  # タグのベクトルインデックス作成
│   └── tag_search.py      # 検索・スコアリングエンジン
├── ui/                    # UIコンポーネント
│   ├── search_bar.py      # 検索窓
│   ├── gallery.py         # グリッド表示
│   ├── viewer.py          # 画像拡大・詳細表示
│   └── drawer.py          # ブックマーク一覧
├── download_assets.py     # 翻訳辞書ダウンローダー
├── index.py               # 初期スキャン・サムネイル生成
└── data/                  # データ保存先 (自動生成)
    ├── images/            # ★ここに検索したい画像を入れてください
    ├── db/                # index.dbが生成されます
    └── thumbnails/        # 縮小画像が生成されます

```

## 🚀 使い方 (初期セットアップ)

初回のインデックス構築手順です。順番にスクリプトを実行してください。

### Step 0: 一括で実行

一括で実行する場合は、以下のコマンドを実行してStep 1~4はスキップしてください。（画像ファイル数によっては数時間かかります）

```bash
.\setup.ps1

```

### Step 1: 辞書のダウンロード

10万語規模の英日翻訳辞書を取得します。（初回のみネット接続が必要です）

```bash
python -m core.download_assets

```

### Step 2: タグリストのベクトル化

AIの出力タグを意味のベクトルに変換し、検索用のインデックスを作成します。（初回のみネット接続が必要です）

```bash
python -m core.vectorize_tags

```

### Step 3: 画像のスキャンとサムネイル生成

`data/images` フォルダ内の画像をデータベースに登録し、表示用の軽いサムネイルを作成します。

```bash
python -m core.index

```

### Step 4: AIによる画像解析（タグ付け）

データベースに登録された画像に対して、AIが自動で内容を判別しタグを付与します。
*(※ 枚数が多い場合は時間がかかります。DirectMLによるGPU支援が効いているかターミナルで確認してください)*

```bash
python -m core.tagger

```

## 🎨 アプリの起動と操作

すべての準備が整ったら、以下のコマンドでアプリを起動します。（完全オフラインで起動します）

```bash
python app.py

```

ブラウザが起動し、ローカルサーバー（デフォルト: `http://localhost:8000`）でUIが表示されます。

### 基本操作

* **検索窓**: 日本語、英語、俗語など好きな言葉を入力してEnterを押します。`-`（ハイフン）を先頭に付けることで**除外検索（マイナス検索）**も可能です。
* **サジェスト**: 文字を入力すると、DB内に存在するタグや辞書のヒット件数がサジェスト表示されます。
* **ギャラリー**: 画面上の**ピンチイン・ピンチアウト**で、直感的に画像の列数（3〜6列）を変更できます。

* **お気に入り**: サムネイル右下、またはビューア内のハートマークで画像を「お気に入り」に登録できます。検索窓が空の状態で検索を実行すると、お気に入り一覧が表示されます。
* **ブックマーク**: 検索窓のブックマークアイコンから、現在の検索条件を保存できます。画面右スワイプでドロワーを開き、保存した条件をすぐに呼び出せます。

### ビューア操作 (画像クリック時)

* **ダブルタップ**: 等倍 ⇔ 2倍ズームの切り替え。
* **ピンチ操作**: 自由な倍率での拡大・縮小とドラッグ移動。
* **スワイプ (左右)**: 前後の画像へ移動します。
* **スワイプ (上)**: 画面下部から「ファイル名」「ファイルパス」「付与されたタグ一覧」の詳細パネルを引き出します。
* **スワイプ (下)**: 詳細パネルを閉じる、またはビューア自体を閉じます。
